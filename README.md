# flutter_app

Flutter課程的作品

## 課程大綱

本次上課的教學是教導學生如何使用Flutter做出一個APP，除了了解Flutter結構外，也會帶我們如何串接API，如何處理API的資料。

## 必要工具

在課程會使用的軟體：
- Andriod Studio
- Flutter SDK
  - 打開 Flutter資料夾裡的 flutter_console 檔案
  - 指令：flutter doctor ，並依循著裡面的訊息去做安裝的動作
- 擴充軟體：Android Studio -> Setting -> Plugins 搜尋並安裝 Flutter、Dart
